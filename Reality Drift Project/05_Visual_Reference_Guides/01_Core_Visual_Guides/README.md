# Visual Reference Guides

A collection of visual explainers and reference guides from the Reality Drift framework.

These documents translate core concepts into concise visual formats designed for rapid understanding, sharing, and reference. They focus on communicating key mechanisms, patterns, and failure modes through diagrams, illustrations, and short-form explanations.

Topics include Synthetic Realness, the Optimization Trap, proxy optimization, metric gaming, semantic drift, AI-generated content, and the broader dynamics of Reality Drift.

The guides are intended to function as conceptual entry points for readers encountering these ideas for the first time while also serving as reusable reference material for researchers, writers, educators, and AI practitioners.

Contents include:

- Synthetic Realness

- Why AI Content Feels Fake

- The Optimization Trap

- When Metrics Replace Purpose

- Framework Maps

- Visual Mechanism Guides
