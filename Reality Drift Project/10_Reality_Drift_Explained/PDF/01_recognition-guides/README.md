# Recognition Guides

## PDF Recognition Guides for the Reality Drift Framework

### Overview

This directory contains PDF editions of the Recognition Guides published through the Reality Drift Explained project.

These short-form guides are designed to help readers identify recurring structural patterns associated with reality drift, optimization pressure, semantic instability, and digitally mediated environments.

Each document begins with recognizable modern experiences and connects them to broader systemic dynamics described throughout the Reality Drift framework.
