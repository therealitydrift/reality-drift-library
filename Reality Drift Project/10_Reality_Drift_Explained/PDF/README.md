# PDFs

## PDF Editions of Reality Drift Explained

### Overview

This directory contains PDF editions of the short explanatory papers and visual explainers published through the Reality Drift Explained project.

The materials in this folder organize the project into grouped PDF collections covering modern questions, recognition guides, drift diagnostics, cross-domain applications, conceptual connections, and visual explainers.

These documents are designed as lightweight, accessible entry points into the broader Reality Drift framework.

## Role Within Reality Drift Explained

The PDF collections provide portable, fixed-format versions of the broader Reality Drift Explained project.

These materials are intended to make structural concepts easier to share, archive, reference, and distribute across different platforms and environments.
