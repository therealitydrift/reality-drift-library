# Published Essays — Reality Drift

This directory contains the complete set of published long-form essays from the Reality Drift project.

Each file is a finalized PDF intended as a stable reference artifact. These essays develop and apply the core Reality Drift frameworks across domains including cognition, language, culture, work, technology, and modern life.

---

## Contents

This folder includes nineteen essays, each addressing a distinct failure mode or structural pattern within high-acceleration symbolic environments, including:

- Filter fatigue and cognitive overload  
- Semantic drift and meaning degradation  
- Synthetic realness and simulated authenticity  
- Optimization traps and metric substitution  
- Temporal dislocation and vapor work  
- The drift principle and systems that persist without correction  
- Language, AI, and recursive compression  
- The “5%” and emergent cognitive stratification  

Each essay stands independently while contributing to a shared conceptual architecture.
