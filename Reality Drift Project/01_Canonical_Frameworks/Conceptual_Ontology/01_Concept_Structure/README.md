# Concept Structure

This folder contains the internal structural mapping of the Reality Drift framework’s core concepts.

These materials organize the framework at the concept level by defining primary concepts, relational nodes, taxonomic structures, fidelity indexing, and machine-readable data exports. They form the first-order conceptual layer beneath the broader ontology.



---

## Structure

- **[Core_Concepts](./Core_Concepts)**  
  Primary conceptual definitions for core framework terms in structured markdown form.

- **[Data_Exports](./Data_Exports)**  
  Machine-readable JSON exports of core concept structures for structured reference and external use.

- **[Drift_Fidelity_Index](./Drift_Fidelity_Index)**  
  Indexing model mapping the relationship between drift intensity and semantic fidelity across framework conditions.

- **[Relationship_Notes](./Relationship_Notes)**  
  Local graph-node documents describing conceptual relationships between primary framework concepts.

- **[Taxonomy](./Taxonomy)**  
  Higher-order taxonomic classification of Reality Drift mechanics and conceptual categories.

---

## Function

These materials serve as the conceptual organization layer of the Reality Drift ontology. They establish how core concepts are structured, related, categorized, and indexed before being formalized into broader ontology frameworks.
