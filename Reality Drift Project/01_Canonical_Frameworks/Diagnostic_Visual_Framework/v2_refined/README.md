# v2_refined — Diagnostic Visual Framework

This directory contains the current canonical visual models of the Reality Drift framework.

These diagrams formalize recurring structural patterns across systems:

- optimization shifting from outcomes to proxies
- constraint weakening and collapse
- representational substitution for material change
- feedback loops that reinforce misalignment

Each image is designed to function as a standalone diagnostic while remaining interoperable with the broader framework. Together, they describe how systems remain operational while gradually losing alignment with the reality they were meant to track.

**What’s different in v2:**

- simplified visual language
- tighter mechanism framing
- consistent naming and structure
- clearer cross-domain applicability

**Scope:**
These visuals span multiple domains including AI, media, institutions, culture, and politics, but are unified by a shared structural dynamic rather than domain-specific explanation.
