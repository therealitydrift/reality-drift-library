# Canonical Frameworks

### Overview

This directory contains the canonical structural frameworks of the Reality Drift project. These materials define the core conceptual architecture of the framework and are treated as stable reference artifacts.

Each subdirectory represents a distinct layer of the canonical framework.

---

### Directory Structure

- Conceptual_Ontology  
  Foundational concepts and structural relationships.

- Lexical_Frameworks  
  Canonical vocabularies, taxonomies, and term definitions.

- Diagnostic_Visual_Framework  
  Visual models used for structural analysis and diagnosis.

- Framework_Atlases  
  Consolidated visual reference atlases of the framework.
