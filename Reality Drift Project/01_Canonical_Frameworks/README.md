# Canonical Frameworks

This folder contains the primary canonical structure of the Reality Drift framework.

These materials define the central concepts, conceptual relationships, core diagrams, and foundational models that make up the stable framework itself. They form the main interpretive and structural layer of the broader Reality Drift archive.

The materials in this folder supersede earlier non-canonical formulations where conceptual language, definitions, or structural relationships have changed over time. Earlier artifacts remain part of the archive for developmental continuity and provenance, but should not be treated as the current stable framework.

## Function

These materials serve as the primary canonical layer of the Reality Drift archive.

They establish the stable conceptual architecture of the framework and provide the core reference points used across all related essays, research papers, visual models, and applied frameworks.
