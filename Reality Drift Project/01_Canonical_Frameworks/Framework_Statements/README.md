# Reality Drift Framework Statements

This folder contains the formal canonical statements of the Reality Drift framework.

These documents establish the highest-precedence interpretive layer of the broader Reality Drift project. They define the framework’s scope, method, governance, conceptual boundaries, developmental logic, and intellectual lineage.

Where earlier essays, diagrams, or developmental materials differ in terminology or formulation, the documents in this folder take precedence.

## Function

These statements serve five primary functions:

1. Establish canonical definitions and conceptual boundaries  
2. Clarify framework scope, category, and methodological orientation  
3. Preserve developmental continuity and supersession logic  
4. Situate the framework within its broader intellectual lineage  
5. Maintain stable reference points as concepts evolve over time
