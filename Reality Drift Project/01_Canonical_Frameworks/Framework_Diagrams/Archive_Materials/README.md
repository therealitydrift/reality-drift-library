# Archived Diagram Materials (README v2)

This folder preserves earlier visual development of the Reality Drift framework.

These materials document the evolution of the framework’s visual architecture before the current canonical diagrams were established. They include earlier framework diagrams, ontology-based visual maps, and simplified infographic illustrations developed across multiple stages of the project.

These materials remain part of the archive for provenance and historical continuity, but they do not supersede the current canonical visual layer.

---

## Structure

- **[01_Framework_Diagrams](./01_Framework_Diagrams)**  
  Earlier generational versions of the core framework diagrams, including legacy, refined, and expanded visual architectures.

- **[02_Ontology_Visuals](./02_Ontology_Visuals)**  
  Earlier ontology-based visual maps showing conceptual relationships, cognitive structures, and broader framework organization.

- **[03_Infographic_Illustrations](./03_Infographic_Illustrations)**  
  Simplified infographic and hand-drawn visual interpretations of the framework designed for broad accessibility and conceptual compression.

---

## Function

These materials preserve the developmental history of the framework’s visual evolution.

They provide historical context for how the conceptual architecture was translated into visual form across different stages of refinement.
