# Wikipedia Definition Visuals

This folder contains simplified visual definition cards for the core concepts of the Reality Drift framework.

These visuals translate canonical concepts into a standardized encyclopedia-style format designed for clarity, fast recognition, and broad accessibility. Each visual presents a concise definition and structural summary of a major framework concept.
