# Early Concept Guides

**README Version: v2**

This folder contains the earliest visual concept guides created for the Reality Drift framework.

These guides represent the first attempts to compress the framework into accessible, portable visual summaries. They preserve the early conceptual language, structure, and visual logic used to introduce the framework before later refinement.



They remain useful as historical entry points into the framework’s early public-facing form.
