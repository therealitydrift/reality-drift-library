# Visual Concept Guides

**README Version: v2**

This folder contains simplified visual reference guides for the core concepts of the Reality Drift framework.

These materials are designed for faster recognition, lighter orientation, and easier conceptual entry. They translate abstract framework concepts into compressed visual forms that prioritize accessibility over full structural depth.

The folders here preserve the progression from early concept guides into more refined and stabilized visual reference systems.

---

## Folder Index

- **[01_Early_Concept_Guides](./01_Early_Concept_Guides/)**  
  First-generation visual concept cards and simplified explanatory diagrams. These preserve the early public-facing visual language of the framework.

- **[02_Refined_Concept_Guides](./02_Refined_Concept_Guides/)**  
  More developed and structurally aligned concept guides reflecting later refinement of the core framework vocabulary.

---

## Function

This folder serves as the visual orientation layer of the framework.

It provides compressed conceptual entry points for readers who want fast recognition of key ideas before moving into the full papers, diagrams, or ontology materials.
