# Cognitive Artifacts

This collection documents the physical and symbolic objects that hold cognition outside the mind. These artifacts represent fragments of memory systems, administrative systems, institutional records, procedural tools, and damaged representational media. 

This visual collection maps how reality becomes translated into forms, labels, diagrams, queues, ledgers, archives, credentials, and checkpoints. Many appear broken, incomplete, misaligned, or abandoned. That condition is intentional.

The collection explores several recurring themes:

- **Externalized cognition** — how thought is stored in objects, paperwork, and systems.

- **Administrative memory** — records as extensions of institutional knowledge.

- **Procedural drift** — systems that remain operational while gradually losing clarity or alignment.

- **Archive decay** — fragments detached from original context.

- **Representation failure** — when the artifact remains but the meaning weakens.

This folder is part of the broader *Reality Drift Framework* visual archive.

## Notes

These artifacts function as conceptual materials within the Reality Drift archive, exploring how modern systems externalize cognition into physical and procedural forms. They focus on what remains when meaning is compressed into systems: fragments, records, credentials, annotations, queues, and damaged traces of coordination.
