# Everyday Interface Overload

## Overview

This directory contains illustrated scenes depicting the hidden friction, procedural burden, interface confusion, and cognitive overload embedded in ordinary modern life.

The images focus on situations in which simple activities become mediated by apps, kiosks, verification systems, digital menus, surveillance infrastructure, delivery systems, and overlapping layers of administrative work.

## Context

These visuals document the texture of modern systems that remain functional while becoming increasingly difficult to navigate.

They illustrate how additional interfaces, procedures, verification layers, and digital requirements transfer more interpretive and administrative work onto the individual.

The collection is part of the broader Reality Drift visual archive.
