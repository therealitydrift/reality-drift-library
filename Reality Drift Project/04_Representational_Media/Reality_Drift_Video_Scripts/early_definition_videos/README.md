# Early Definition Videos

**README Version: v1**

This folder contains early video scripts and structured explainer materials for the core Reality Drift concepts. These were created during the initial public-definition phase of the framework and were intended to establish the basic vocabulary in a more accessible spoken format.

They reflect an earlier stage of conceptual articulation and may differ slightly from later canonical formulations.

---

## Video Index

- [Filter Fatigue Explained](./filter-fatigue-explained/)  
  Early explainer on cognitive overload, attention fragmentation, and infinite choice structures.

- [Optimization Trap Explained](./optimization-trap-explained/)  
  Early articulation of how optimization can degrade the very thing it was meant to improve.

- [Reality Drift Explained](./reality-drift-explained/)  
  Foundational overview of how systems remain functional while losing alignment with reality.

- [Synthetic Realness Explained](./synthetic-realness-explained/)  
  Early exploration of simulated authenticity and representational substitution.

---

## Function

These materials serve as:

- Introductory public-facing explainers  
- Early teaching tools for framework vocabulary  
- Video-first translations of core definitions  
- Historical artifacts of conceptual stabilization  

They prioritize clarity and accessibility over precision.

---

## Archive Note

These are early working definition videos. Some terminology, framing, and boundaries have since been refined in the canonical framework library. Where differences exist, the canonical concept papers should be treated as the primary reference.
