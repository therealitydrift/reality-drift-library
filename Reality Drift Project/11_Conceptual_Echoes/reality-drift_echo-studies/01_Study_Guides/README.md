# Study Guides

**README Version: v2**

This directory contains early exploratory study guides from the Reality Drift project.

These materials were developed during an experimental phase in which Reality Drift concepts were translated into structured learning formats, guided exercises, and conceptual teaching tools.

They should not be understood as formal educational materials, accredited curricula, or finalized framework documents.

Their primary purpose was to test whether complex conceptual structures could be made teachable, transferable, and repeatable across different interpretive environments.

---

## Scope

The materials in this directory include:

- guided conceptual walkthroughs  
- structured examples and flashcard systems  
- glossary-based learning scaffolds  
- early explanatory models for Reality Drift and related concepts  
- experimental meaning frameworks used for conceptual reinforcement  

These documents were used as internal developmental tools rather than finalized public-facing resources.

---

## Archive Status

This directory is preserved as part of the early working archive. It reflects a phase of conceptual experimentation where educational framing, study systems, and instructional structures were used as speculative tools for testing the portability of Reality Drift across learning contexts.

Some materials may contain:

- provisional terminology  
- incomplete definitions  
- outdated conceptual relationships  
- simplified instructional framing  
- experimental educational structures later revised or abandoned  

These documents are retained for developmental and historical context only. Where differences exist, later canonical framework materials take precedence.
