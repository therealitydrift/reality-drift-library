# Reality Drift: Core Drift Types

This collection outlines the primary structural pathways through which systems lose alignment with reality while continuing to operate.

Each document defines a specific drift pattern observed across organizations, institutions, and AI systems. These are atomic components of the broader Reality Drift framework.

---

## Core Drift Types

- [Bureaucratic Drift: When Processes Expand and Outcomes Stall](./bureaucratic-drift-process-expansion-inefficiency-organizations.pdf)  
  Systems become process-heavy while real-world results degrade.

- [Customer Service Drift: When Support Systems Stop Solving Problems](./customer-service-drift-why-support-systems-fail-to-resolve-problems.pdf)  
  Responsiveness increases while actual problem resolution declines.

- [Incentive Drift: When Rewards No Longer Align With Outcomes](./incentive-drift-rewards-vs-outcomes-misalignment-systems.pdf)  
  Systems reinforce behavior that does not produce real-world impact.

- [Narrative Drift: When Explanations Detach From Reality](./narrative-drift-detached-explanations-from-reality-reality-drift.pdf)  
  Coherent stories persist even as they lose grounding in actual conditions.

- [Institutional Drift: When Organizations Lose Alignment With Reality](./reality-drift-institutional-drift-organizational-misalignment.pdf)  
  Systems remain operational but become disconnected from external conditions.

- [Temporal Drift: The Collapse of Shared Time in Modern Systems](./temporal-drift-collapse-of-shared-time-modern-systems.pdf)  
  Systems lose synchronization, fragmenting shared context and continuity.

---

## Framework Context

These drift types represent recurring failure modes across modern systems:

- Organizational systems optimizing for internal processes over outcomes  
- Metrics improving while real-world impact declines  
- Language and narratives maintaining coherence without grounding  
- Incentives producing predictable but misaligned behavior  
- Time and context fragmenting across distributed environments  

Reality Drift describes this broader pattern: systems continue to function, but gradually lose alignment with the reality they are meant to represent or respond to.
