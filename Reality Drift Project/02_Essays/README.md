# Essays

**README Version: v1**

This folder contains the essay layer of the Reality Drift project.

This section preserves the broader written ecosystem surrounding the framework: published essays, developmental drafts, archival notes, and thematic maps.

These writings document how the framework was explored, tested, expanded, and publicly articulated over time.

Some essays function as direct explanatory extensions of the framework. Others preserve earlier language, alternate framings, or partial formulations that were later revised.

Where conceptual conflicts exist, the materials in `01_Canonical_Frameworks` supersede these essays.
