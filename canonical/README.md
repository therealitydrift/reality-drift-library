# Canonical Concepts

This directory contains the core conceptual definitions of the Reality Drift framework.

The concepts focus on:

- semantic degradation
- optimization failure
- cognitive overload
- representational drift
- system-level loss of grounding
- AI-mediated symbolic environments

Author: A. Jacobs (2023–2026)
